# Hostel Allocation Database System

## Overview

Hostel Allocation Database System is a semester project developed to simplify hostel management through a centralized desktop application. The system helps manage room allocation, student and guest records, staff information, bookings, payments, and reports using an integrated SQL Server database.

This project was developed using C#, .NET Framework, Windows Forms, and Microsoft Visual Studio with a focus on providing an organized and user-friendly hostel management solution.

---

## Features

- Student Registration and Management
- Guest Record Management
- Staff Management
- Room Allocation
- Room Booking System
- Payment Management
- Report Generation
- User-Friendly Desktop Interface
- SQL Server Database Integration

## How to Run

1. Clone or download this repository.
2. Open the solution in Microsoft Visual Studio.
3. Restore or create the SQL Server database.
4. Update the database connection string in `App.config` if required.
5. Build and run the project.
